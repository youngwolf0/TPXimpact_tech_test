ABOUT tech_test_theme
------------

tech_test_theme is the default theme for Drupal 8.  It is a flexible, recolorable theme
with a responsive and mobile-first layout, supporting 16 regions.

The tech_test_theme theme is named after Jean tech_test_theme, one of the original programmers
for the ENIAC computer.

This theme is not intended to be used as a base theme.

To read more about the tech_test_theme theme please see:
https://www.drupal.org/docs/8/core/themes/tech_test_theme-theme
